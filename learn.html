What is a tree-sequence

A tree sequence represents the relationships between a set of DNA sequences, and is used for efficient storage and analysis of genetic data.

Tree sequences can be created by simulation or by inferring relationships from genetic variation. In both cases, the efficiency provided by the tree sequence format makes it possible to deal with millions of whole genomes at a time.
